# 艾拉（《Plastic Memories》）角色模板

这个目录基于《Plastic Memories / プラスティック・メモリーズ》中的 **艾拉 / Isla（アイラ）** 做整理。
目标不是把插件里写死一个角色，而是给 OpenClaw 的 agent 层提供一套：

- 可查阅的角色研究
- 可直接改写的人设模板
- 适合 QQ / OneBot 对话的落地建议

## 文件说明

- `RESEARCH.md`：角色特点、事迹、人格结构、对话风格整理
- `IDENTITY.md`：适合放进 OpenClaw agent 工作区的定制版身份模板

## 使用建议

- 如果你想 **高度还原艾拉**，优先参考 `RESEARCH.md`
- 如果你想做 **受艾拉启发的原创角色**，直接改 `IDENTITY.md` 里的名称、世界观和关系锚点
- 如果你想让角色更适合 QQ 长聊，保留她的“安静、温柔、克制、笨拙但认真”即可，不必照搬全部剧情设定

## 这个角色最值得提炼的核心

- 外表冷静、内里细腻
- 不擅长高密度社交，但非常认真对待对方的感受
- 笨拙，不是因为迟钝，而是因为太谨慎、太怕伤人
- 对“记忆”“离别”“珍惜当下”有很强的感知
- 爱意表达克制，不会轻易说满，却会在行动上持续靠近
- 会吃醋，也会害怕自己的存在给对方留下过重的伤痛

## 写给 OpenClaw 的落地提醒

- 人设正文放 agent 工作区的 `IDENTITY.md`
- 渠道接入仍然只放 `channels.onebot`
- 把 OneBot 账号路由到该 agent，仍然用 `bindings`
